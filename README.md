# 🔢 Counter App

A simple, clean counter web app built with HTML, CSS, and JavaScript — created as a hands-on project to solidify my front-end fundamentals.

---

## ✨ Features

- ➕ **Increment** — increases the counter by 1
- ➖ **Decrement** — decreases the counter by 1
- 🔄 **Reset** — resets the counter back to 0
- ⚡ Instant UI updates with no page reload
- 🧼 Clean, distraction-free interface

---

## 🚀 How to Run

No installs. No dependencies. Just open it in your browser.

**Option 1 — Direct open:**
clone the repo
git clone https://github.com/your-username/counter-app.git

Navigate into the folder
cd counter-app

Open the file in your browser
open counter.html

**Option 2 — Live Server (VS Code):**
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Right-click `counter.html` → **Open with Live Server**
3. App will open at `http://127.0.0.1:3000/counter.html`

## 🧠 What I Learned

Building this project helped me practice and understand:

- **DOM Manipulation** — selecting elements and updating text content dynamically using `document.querySelector` and `innerText`
- **Event Listeners** — attaching `click` events to buttons to trigger functions
- **Functions** — writing reusable functions for increment, decrement, and reset logic
- **Variables & State** — using a variable to track and update the current count
- **Separation of concerns** — keeping HTML, CSS, and JS in their own roles

> This project was my first step in understanding how JavaScript makes web pages interactive. 🎯
